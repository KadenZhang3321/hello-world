# hello-world

这是一个包含多个Java程序的学习项目。

## 项目文件说明

### 📝 LetterCounter.java
**字符串字母统计程序**
- **功能**: 输入一个字符串，统计并输出每个字母出现的次数
- **特性**:
  - 自动转换为小写统计（忽略大小写）
  - 只统计字母字符，忽略数字和特殊符号
  - 按字母顺序显示统计结果
  - 支持交互式输入，可连续统计多个字符串
- **使用方法**:
  ```bash
  javac LetterCounter.java
  java LetterCounter
  ```
- **示例输入**: `Hello World!`
- **示例输出**: 
  ```
  字母 'd': 1 次
  字母 'e': 1 次  
  字母 'h': 1 次
  字母 'l': 3 次
  字母 'o': 2 次
  字母 'r': 1 次
  字母 'w': 1 次
  ```

### 🔢 hellp.java
**进制转换程序**
- **功能**: 将十进制数字转换为指定进制（2-36进制）
- **特性**: 支持小数的进制转换
- **使用方法**: 从标准输入读取数字和目标进制

### 👋 hello.java  
**字符串比较演示程序**
- **功能**: 演示Java中字符串比较的不同方法
- **内容**: 包含 `compareTo()`, `compareToIgnoreCase()` 等方法的使用示例

## 运行环境
- Java 8 或更高版本
- 支持UTF-8编码的终端（用于正确显示中文）
